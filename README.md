## Run Locally

Clone the project

```bash
  git clone https://github.com/justshuvo/areacalc.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start
```


## Tech Stack

**Client:** HTML, CSS

**Server:** Node, Express
